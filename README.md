# Restaurant Admin Dashboard

## Features
- Menu CRUD operations
- Search with debouncing
- Order management
- Optimistic UI updates
- MongoDB aggregation

## Tech Stack
React, Node.js, Express, MongoDB

## Setup
cd server
npm install
npm run dev

cd client
npm install
npm start

## API Endpoints
GET /api/menu
POST /api/orders
PATCH /api/menu/:id/availability
